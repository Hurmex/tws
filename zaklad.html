K vytváření webových stránek se používá jazyk HTML = Hyper Text Markup Language, skládající se z elementů, které "říkají" prohlížeči jak má stránka vypadat.

Na začátku webových stránek, přesněji v jeho zdrojovém kódu, se nachází <!DOCTYPE html>, což je typ dokumentu, tento je HTML5.

Ve zdrojovém kódu stránky se také nachází elementy jako <html>...</html>, <head>...</head>, <title>...</title>, <body>...</body>, <p>...</p>, nebo <h1-h6>...</h1-h6>

<html>...</html> je hlavním elementem HTML stránky

<head>...</head> obsahuje informace o HTML stránce, neviditelný obsah

<title>...</title> je titulek stránky

<body>...</body> je tělo dokumentu, viditelný obsah

<p>...</p> je odstavec

<h1-h6>...</h1-h6> je nadpis, který se liší velikostmi s tím, že h1 je největší a h6 naopak nejmenší

Přípona HTML souborů je .html a úvodní stránka by se měla nazývat index.html
